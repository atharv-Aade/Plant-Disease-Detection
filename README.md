# 🌱 Plant Disease Detection Using IoT

An IoT-based smart agriculture system that monitors plant environmental conditions and detects plant diseases using Machine Learning. The system collects real-time sensor data, automates farm operations, and identifies plant health status using an AI model.

---

## 📌 Project Overview

Plant Disease Detection Using IoT is a smart farming solution designed to help farmers monitor crop conditions and detect diseases at an early stage.

The system uses IoT sensors to collect environmental data such as temperature, humidity, and soil moisture. Based on the collected data, automated actions are performed using a microcontroller. A Machine Learning model is integrated to analyze plant images and classify them as healthy or infected.

---

## 🚀 Features

- 🌡️ Real-time temperature monitoring
- 💧 Soil moisture monitoring
- 💨 Humidity monitoring
- 🌿 Plant disease detection using AI model
- 📷 Disease detection through webcam/image upload
- 🤖 Machine Learning-based plant health classification
- ⚡ Automatic irrigation control
- 🌬️ Automatic fan control
- 🏠 Automatic roof control
- 📊 Live sensor data visualization
- 📱 IoT monitoring using Blynk platform
